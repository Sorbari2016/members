# Members only

## Description:

Members only is a simple web application for a fictitious clubhouse, where members can write anonymous posts. The guildelines of the project is designed to help understand and know to implement:

- Role-Based Access Control (RBAC) feature.
- Validation & Sanitation of form data.
- Database integration, &
- Authentication of node js app.

## Features:

## Tech Stack:

- Node JS + Express JS
- EJS, & HTML
- CSS
- Postgres
